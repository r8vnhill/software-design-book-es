\subsection{\textit{MacOS}}
  \index{MacOS}

  Para instalar \textit{Kotlin} en \textit{OSX} se pueden seguir las mismas instrucciones que se
  dieron para \textit{Linux}, o se puede utilizar \textit{Homebrew}\index{Homebrew}:

  \begin{bash}
    brew update
    brew install kotlin
  \end{bash}

  Por último comprobamos la instalación:

  \begin{bash}
    kotlinc -version
  \end{bash}
